# Marketing Website UI Kit — The Pack House

Hi-fi recreation of the brand's marketing surface. Heritage leather-goods retailer aesthetic: cream paper, brass accents, deep walnut, generous serif display type.

## Files
- `index.html` — Full single-page composition: nav → hero → category strip → featured product → field-notes journal entry → footer. Click "Add to cart" to see the slide-in cart drawer.
- `Nav.jsx`, `Hero.jsx`, `Categories.jsx`, `Featured.jsx`, `Journal.jsx`, `Footer.jsx`, `CartDrawer.jsx` — modular sections.

## Conventions
- Cream (`#F6F1E9`) is the page wash; ink (`#111009`) only appears full-bleed in the footer.
- Buttons are sharp rectangles, 2 px radius. Never pill, never gradient.
- Every product image is a placeholder — drop in real photography by replacing `.product-img` backgrounds.
- Prices in Vietnamese Đồng (₫), comma-grouped, e.g. `2,800,000 ₫`.
